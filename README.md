# palette-search

Find the closest color in a theme for any given color.

## Usage

```
// theme.json
{
	"red": "#ff000",
	"maroon": "#800000",
	"dark-maroon": "#660000",
	"pink-maroon": "#984C58"
}

npx color-palette #9e3333 theme.json
```
